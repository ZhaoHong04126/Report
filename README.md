# 🔒 專屬密碼管理器 (Secure Password Manager) - 桌面 GUI 版

這是一款專為本機安全儲存帳號與密碼設計的現代化桌面應用程式 (GUI App)。系統整合了 **AES 加密**、**動態防螢幕截圖**、以及最新的 **AI 鏡頭人臉偵測安全防護**，全方位保障您的數位隱私與帳密安全。

---

## 🌟 核心特色功能

### 1. 🎨 現代化 GUI 介面與主題切換
* 採用 **CustomTkinter** 設計，具備高質感圓角、滑動動畫與響應式佈局。
* 支援隨作業系統自動切換的 **深色模式 (Dark Mode)** 與 **淺色模式 (Light Mode)**，或於設定頁面手動鎖定主題。

### 2. 🛡️ 進階安全加密機制
* **Fernet 加密保護**：所有寫入本機 `passwords.txt` 的檔案內容皆採用強健的加密算法，

### 3. 🚫 動態螢幕防截圖功能 (Windows)
* 整合 Windows `SetWindowDisplayAffinity` 系統 API。
* 當您點進「**所有密碼清單**」或彈出「**驗證/編輯對話框**」時，防截圖機制會**自動開啟**。此時，任何螢幕截圖軟體（如 Windows 內建剪取工具）、螢幕錄影、遠端連線或視訊投影軟體（如 OBS, Zoom, Teams, Discord 畫面分享）將完全無法擷取本視窗內容（畫面會呈現全黑或隱形），完美防止密碼被暗中翻拍或記錄。

### 4. 🔑 二次主密碼驗證
* 密碼清單上所有的敏感按鈕（包含「顯示/隱藏密碼 👁️」、「複製帳號」、「複製密碼」、「編輯 ✏️」與「刪除 ❌」），點擊後皆會彈出安全驗證視窗，需輸入系統主密碼方可執行，防止他人趁您短暫離開電腦時操作。

### 5. 📷 攝影機防旁窺與離座自動上鎖
* **操作時防窺檢測**：在進行上述二次主密碼驗證時，App 會啟動相機檢測周邊環境：
  * 👤 **安全通過**：鏡頭前**僅有 1 人** (使用者本人) 時，通過驗證。
  * 🚨 **旁窺攔截**：鏡頭前偵測到 **2 人以上** 時，App 會**立刻發出安全警報並攔截該操作**，防止背後有人偷窺。
  * ⚠️ **無人驗證**：偵測到 **0 人** 時，提示使用者正對鏡頭重試。
  * 💻 **無相機 Fallback**：若您的電腦無裝設攝影機，系統會自動退回到單純的主密碼驗證，確保功能相容性。
* **顯示時離座自動隱藏**：當您在清單中將密碼展示為明文後，背景會開啟**多執行緒非同步監控**（不卡頓 UI 介面），每 3 秒自動巡檢一次人臉：
  * 當您離座（人臉數變為 **0**）或背後有人湊過來（人臉數變為 **2** 以上），App 會**瞬間將畫面上所有明文密碼變回遮罩星號 `*` 隱藏**，並發出提示！當無明文顯示時相機將自動關閉，節省電腦電力。

### 6. 🎲 隨機密碼生成器
* 在新增帳密分頁中，提供一鍵生成「16 位高強度隨機密碼（含大小寫英文字母、數字及特殊符號）」的功能，並自動帶入輸入框。

## 🚀 如何安裝與運行

### 預置條件
1. 已安裝 [Python 3.10+](https://www.python.org/)。
2. 啟動您的專案虛擬環境：
   ```powershell
   .venv\Scripts\activate
   ```

### 安裝必要依賴套件
```powershell
pip install customtkinter cryptography opencv-python pyinstaller
```

### 運行 GUI 應用程式
```powershell
python gui.py
```

### 編譯並打包成獨立 `.exe` 執行檔
如果您想在其他無 Python 環境的 Windows 電腦上單擊執行，可以透過以下指令將專案編譯為單一執行檔：
```powershell
pyinstaller --noconsole --onefile --name "專屬密碼管理器" gui.py
```
打包完成後，您可以在專案目錄下的 `dist` 資料夾內找到 `專屬密碼管理器.exe`。

## 📂 檔案目錄結構描述
* `gui.py`：本 App 的主程式碼（內含密碼學邏輯、CustomTkinter GUI 介面、相機人臉偵測、動態防截圖）。
* `main.py`：原備份的命令列介面 (CLI) 程式。
* `passwords.txt`：加密儲存的密碼文字檔案（若路徑 `C:\Users\USER\OneDrive\Desktop\Report\passwords.txt` 不存在，將預設建立於本目錄下）。
* `dist/專屬密碼管理器.exe`：打包好的 Windows 獨立執行檔。
